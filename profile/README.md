# 🥔 POTATO (포타토)
> **오늘의 요리를 기록하고, 나만의 감자 캐릭터를 빌드업하는 요리 소셜 플랫폼**

<br>

## 🌟 프로젝트 개요
**POTATO**는 사용자가 매일의 요리 활동을 인증하고, 그 보상으로 캐릭터를 본인만의 스타일로 커스터마이징하며 성장시키는 서비스입니다. <br>캐릭터의 외형 변화를 통해 요리 습관의 성취를 시각적으로 즐길 수 있는 환경을 제공합니다.

<br>

## 🏗️ Information Architecture
전체 서비스는 사용자의 활동 흐름에 따라 유기적으로 연결되어 있습니다:

| 대분류 | 주요 기능 및 화면 |
| :--- | :--- |
| **인증 (Auth)** | 로그인, 회원가입, 아이디/비밀번호 찾기, 스플래쉬 화면 |
| **메인 (Home)** | 캐릭터 상태 확인, 요리 인증 버튼, 레벨 정보 및 요리 기록 달력 |
| **상점 (Shop)** | 헤드웨어, 의상, 액세서리, 리미티드 에디션 운영 |
| **커뮤니티** | 친구 피드, 레시피 추천, 리워드 시스템 연동 |

<br>

## 🎨 Main Features

### 1. 캐릭터 커스터마이징 & 빌드업 👗
- 퍼스널라이징: 상점에서 구매한 아이템을 인벤토리에서 착용하여 캐릭터의 외형을 실시간으로 변경합니다.
- 리워드 경제: 요리 인증 및 광고 시청을 통해 '스푼(Spoon)'을 획득하고 아이템을 수집합니다.

### 2. 요리 아카이빙 & 소셜 📸
- 성취 기록: 누적 인증 수에 따라 레벨 배지가 부여되며, 진행 상태를 바(Bar) 형태로 시각화합니다.
- 인증 히스토리: 달력 UI를 통해 과거 요리 사진과 기록을 날짜별로 한눈에 관리합니다.

<br>
